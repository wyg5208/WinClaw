================================================================
  测试注意事项
================================================================

1. 日程管理中的"明天下午2点"等自然语言时间，需要 LLM 正确
   解析为 YYYY-MM-DD HH:MM:SS 格式后传给工具

2. 天气查询依赖和风天气 API Key（环境变量 QWEATHER_API_KEY），
   未配置时会降级为 Web 搜索

3. 知识库测试中的文件路径请根据实际环境替换

4. 记账金额以浮点数传入（如 35.5），内部以分为单位存储

5. 日记的心情参数限定为: happy/sad/neutral/excited/anxious/calm/stressed
   天气参数限定为: sunny/cloudy/rainy/snowy/windy/foggy
   LLM 需要将用户自然语言（如"开心""晴天"）映射为对应的英文值

6. 对于更新/删除操作，需要先从前面的创建或查询中获取到对应的 ID
   测试时请按顺序执行，确保 ID 存在

7. 聊天历史搜索基于 history.db，需要有之前的对话记录才能返回结果
